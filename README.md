# 🎓 EDUC_AI

**Plataforma Educativa Inteligente y Gamificada**

EDUC_AI es un entorno de aprendizaje inmersivo (SaaS) de nueva generación. Utiliza Inteligencia Artificial para transformar el contenido tradicional en experiencias dinámicas, adaptándose automáticamente a distintos estilos de aprendizaje y fomentando la participación mediante sistemas de gamificación (XP y Rankings).

## ✨ Características Principales

- **Múltiples Mundos de Aprendizaje:** Contenido adaptado visual, auditiva y teóricamente usando IA (Gemini).
- **Gamificación Integrada:** Sistema de experiencia (XP), misiones y "Salón de la Fama" para motivar a los estudiantes.
- **Centro de Comando Administrativo:** Panel de control en tiempo real para gestionar docentes, alumnos y materias.
- **Prevención de Deserción:** Análisis predictivo de riesgo estudiantil para intervención temprana.

## 🛠️ Tecnologías

- **Frontend:** React 18 + Vite
- **Backend & Auth:** Firebase (Firestore, Auth, Storage)
- **Motor de IA:** Google Gemini 1.5 Flash
- **Diseño UI/UX:** Rotceh Design (Cyberpunk / Glassmorphism)
